## 📘 使用指南

### 下載安裝、開啟程式與更新
- 下載安裝請參考YT影片
- 正確方式開啟程式、程式更新
  1. (無論是否已安裝過，請先執行 setup 進行安裝或修復。)
  2. 確保防毒軟體將 DanceGPT 設為信任；攔截廣告軟體將 DanceGPT 設為允許。
  3. 請雙擊原下載的解壓縮包中的 DanceGPT.bat 開啟程式，切勿使用setup安裝路徑中的檔案開啟（不再需再設定系統管理員開啟，也請勿直接點擊 TySyLdAsTsOpMaAcReSx.exe 開啟程式）
  4. 當登入介面出現時，輸入帳號和密碼進行登入
  5. 更新時請勿覆蓋舊檔，應另行建立一個資料夾並重新執行上述步驟。可以將舊設定移到新資料夾中，檔案包括UserData資料夾、Script_main.m和Script_body.m。

### 前置設定 (<span style="color:red;">⚠️注意: 若無照此設定將導致功能異常</span>)

- 遊戲解析度設定為 1366x768。
- 小地圖置於左上角並顯示地圖名稱；關閉 BUFF 自動排序，讓 BUFF 狀態列可見。  
- Windows 縮放比例設定為 100%。可以在 Windows 的「系統 - 顯示器 - 比例」中設置。  
- 關閉所有遊戲外置聊天視窗。  
- 🔌 使用 Arduino Leonardo  
  - 必須使用 **Arduino Leonardo**（Uno / Mega 不支援）。（蝦某、某皮購買 200 元左右）  
  - Q&A  
    - 換角色或電腦時，Arduino 需要重新燒錄嗎？  
      ➤ 不需要。  
    - 為什麼選擇 Arduino 時會跳回 Virtual？  
      ➤ 可能是因為在程序運行到一半才切換，導致 USB 連線不穩。  
      ➤ 啟動時就進行切換，Arduino 可能沒有成功燒錄或 COM port 選擇錯誤。  
    - 為什麼顯示 avrdude.exe 或 avrdude.conf 路徑錯誤？  
      ➤ 確認路徑中沒有中文字符。  
      ➤ 使用 Arduino 預設的安裝路徑，理論上只需修改 UserName。  
    - 為什麼重啟 Arduino Leonardo 後沒有看到 bootloader？  
      ➤ 開啟系統音效，確認是否有插入或拔出的音效提示。  
      ➤ 在裝置管理員中，重啟後立即點擊「掃描硬體變更」。  

### 功能說明
- #### ⏳ 腳本錄製
  - 請參考YT影片 
- #### 🎯 腳本直用
  - 使用教學流程
    - 1. 挑選職業與等級區間地圖，下載對應腳本
      - 根據你的職業與練功地圖，下載對應的「直用腳本」。
    - 2. 放置必要檔案
      - 將 Direct_Script_main.m 和 body.m 放入 Dance 資料夾中。
      - 將 Direct_setting.txt 放入 UserData 資料夾中。
    - 3. 設置快捷鍵
      - 將遊戲中的快捷鍵設置成與 KeyboardShortcuts.png 圖片中的配置一模一樣。
    - 4. 設定常駐技能（若有）
      - 如果 Direct_setting.txt 中有設定常駐技能，請確保該技能的快捷鍵：
      - 有出現在遊戲右下角的快捷鍵顯示介面上。
      - 使用 Gen_Img 腳本生成對應技能的圖檔。
      - 將產生的圖檔放入 Image_customer 資料夾中。
    - 5. 啟動腳本
      - 開啟 Dance/Direct_Script_main.m，即可開始快樂跳舞！
- #### 🧙‍♂️ 解地圖輪  
  - **Q&A：**  
    - 解輪成功後卻顯示 FAIL 並停止？  
      ➤ 檢查地圖輪秒數是否歸零，UI 是否擋住 CD 圖示。  
    - 解輪的成功率很低？  
      ➤ 在格蘭蒂斯，解輪成功率可達 95%。私服圖像可能調整過以至於無法解輪。
- #### 🪄 輔助技能  
  - **使用說明**：  
    ➤ 適用於 CD 時間固定的技能或消耗品（例如：經驗券）。 
    ➤ 第一個數字代表技能的冷卻時間，即每次施放之間需間隔的時間。
    ➤ 第二個數字為技能的後搖時間，即技能施放完畢後需要等待多久才能行動。
    ➤ (每個按鍵均可單獨設定，並且在運作時會同時生效)  

- #### 🛡️ 常駐技能  
  - **使用說明**：  
    ➤ 適用於技能圖示會在可施放時亮起的技能（例如：CD 時間不固定或會跳無冷卻時間的技能）
  - **設置步驟**：  
    - 1. 將常駐技能圖示放置於快捷鍵位置。
    - 2. 點擊「Gen Img」按鈕。（如果產生的圖示有偏移，請參考必要設定進行調整）
    - 3. 在路徑 \DanceGPT\Image_customer\可用常駐技能 中選擇常駐技能的圖片，並將其移動到 \DanceGPT\Image_customer 目錄下。
    - 4. 重啟 DanceGPT，確認是否成功讀取對應數字的圖檔。
    - 5. 為每個圖檔設定對應的快捷鍵。
    - 6. 設定技能效果的持續時間。

- #### 🚨 測謊偵測  
  - **Q&A 常見問題：**  
    - 為什麼沒有偵測到菲歐娜測謊(or 滑鼠跟蹤測謊)？  
      ➤ 兩個檢查點的間隔應 < 10 秒，否則可能錯過檢查點的測謊檢查。  
    - 為何頻繁遇到測謊？  
      ➤ 必須使用 Arduino Leonardo。非危險時段 (02:00 ~ 08:00) 可以持續跳滿 12 小時以上。  
      ➤ 增加腳本的隨機度和複雜度。  
    - 晚上較容易遇到測謊？  
      ➤ 凌晨 02:00–08:00 期間應避免跳舞，亦避免此時調整腳本。  
    - 平常都正常，今天卻一直收到測謊？  
      ➤ 解完測謊後重啟電腦。仍無改善請隔日再嘗試。

- #### 🔴 紅點偵測  
  - 紅點數量 ≥ N 且超過 T 秒即回村並停止腳本。
 
- #### 🗺️ 換圖結束  

- #### ⏳ 執行時間  
  - 可設定腳本自動停止時間，例如 10 hr 10 min 會在上午 10:10 停止腳本。

- #### ⚙️ 腳本設定  
  - 為什麼會跳出「請將小地圖打開（需顯示地圖名稱，並放置於遊戲視窗的左上角）。目前僅支援遊戲解析度為 1366x768 下運行。」的視窗？
    ➤ 請確認「前置設定」。
  - 解輪後角色位置亂掉了怎麼辦？
    ➤ 請在腳本中設置角色位置重置操作，例如撞牆或在平台上衝刺。 
  - 為什麼腳本運行中會發出嘟嘟嘟的聲音？
    ➤ 這是為了防止角色卡在繩子上。如果角色位置長時間沒有變化，會觸發這個聲音。
  - Image_Capture 資料夾中的圖片有什麼用途？
    ➤ 這些圖片記錄了遊戲運行的狀態，以便在出現問題時進行分析。
