## 📘 使用指南

### 🧰 安裝
1. **安裝**：第一次使用或更新程式，請先執行 `setup` 以進行安裝或修復。  
   請參考YT影片進行下載與安裝。

### 🖥️ 開啟程式
1. **防毒與攔截廣告軟體設置**  
   - 確保您的防毒軟體將 **DanceGPT** 設為信任。
   - 將任何攔截廣告軟體設為允許 **DanceGPT**。

2. **正確開啟程式**  
   - 請雙擊原下載的解壓縮包中的 `DanceGPT.bat` 以開啟程式。
   - 請勿使用 `setup` 安裝路徑中的檔案開啟程式。
   - 不再需要系統管理員權限開啟，也請勿直接點擊 `TySyLdAsTsOpMaAcReSx.exe` 開啟程式。

### ⬆️ 更新程式
1. **更新方式**  
   - 請勿以覆蓋舊檔的方式進行程式更新。
   - 若需要保留舊設定，您可以將以下檔案從舊版本的資料夾移至新版本的資料夾中：
     - `UserData` 資料夾
     - `Script_main.m`
     - `Script_body.m`

## 🛠️ 前置設定 (⚠️注意: 若無照此設定將導致功能異常)

1. **Windows 縮放比例設置**
   - 將 Windows 的縮放比例設為 **100%**。  
     可在 Windows 的「系統 > 顯示器 > 比例」中設置。

2. **遊戲解析度設定**
   - 設定遊戲解析度為 **1366x768**。

3. **小地圖與 BUFF 顯示設置**
   - 小地圖應置於 **左上角**，並顯示地圖名稱。
   - 關閉 **BUFF 自動排序**，讓 **BUFF 狀態列** 可見。

4. **關閉外置聊天視窗**
   - 關閉所有遊戲外置的聊天視窗。

5. 🔌 **使用 Arduino Leonardo**
   - 必須使用 **Arduino Leonardo**（Uno 或 Mega 不支援）。  
     ※ 可以在蝦某、某皮等網站以約 **200~300 元** 購買。

6. **常見問題**

   - **換角色、腳本或電腦時，Arduino 需要重新燒錄嗎？**  
     ➤ 不需要！

   - **為什麼選擇 Arduino 會跳回 Virtual？**  
     ➤ 當按下 Dance 時 UI 跳回 Virtual，表示 Arduino 可能未成功燒錄或選錯 COM port。  
     ➤ 如果跳舞過程中途跳回 Virtual，則可能是 USB 連接不穩或接觸不良。

   - **為什麼顯示 `avrdude.exe` 或 `avrdude.conf` 路徑錯誤？**  
     ➤ 確認路徑中沒有中文字符。  
     ➤ 使用 Arduino 預設的安裝路徑，理論上只需修改 UserName。

   - **為什麼重啟 Arduino Leonardo 後沒有看到 bootloader？**  
     ➤ 開啟系統音效，確認是否有插入或拔出的音效提示。  
     ➤ 開啟裝置管理員觀察，重啟後立即點擊「掃描硬體變更」。

## 功能說明

### ⏳ 腳本錄製
- 請參考 YT 影片。

<br>

### 🗡️ 腳本直用
#### 使用說明
職業與等級符合即可無腦使用，經長時間測試能穩跳 6 小時以上。

#### 設置步驟
1. 挑選對應職業與等級區間的地圖，下載對應腳本。
2. 放入挑選的職業等級腳本與設定檔：
   - `Direct_Script_main.m` 和 `Direct_Script_body.m` 放入 Dance 資料夾。
   - `Direct_setting.txt` 放入 `UserData` 資料夾。
3. 設置遊戲快捷鍵：根據 `KeyboardShortcuts.png` 配置設置。
4. 設定常駐技能（若有）：
   - 根據 `Direct_setting.txt`，確保常駐技能的快捷鍵出現在遊戲右下角的快捷鍵顯示介面上。 
   - 使用 `Gen_Img` 生成技能圖檔並放入 `Image_customer` 資料夾。(可參考常駐技能-設置步驟)
5. 啟動腳本，開始快樂跳舞！

<br>

### 🧙‍♂️ 解地圖輪
#### 常見問題
- **解輪後顯示 FAIL 並停止**：檢查地圖輪秒數是否歸零，BUFF列是否遮擋。請依照「前置設定」的說明進行調整。
- **解輪成功率低**：格蘭蒂斯成功率可達 95%，私服可能有圖像調整，導致無法解輪。

<br>

### 🪄 輔助技能
#### 使用說明
適用於冷卻時間固定的技能或消耗品（例如：經驗券）。  
第一個數字代表技能的冷卻時間，即每次施放之間需間隔的時間。  
第二個數字為技能的後搖時間，即技能施放完畢後需要等待多久才能行動。  
每個按鍵均可單獨設定，並且在運作時會同時生效。

<br>

### 🛡️ 常駐技能
#### 使用說明
適用於技能圖示會在可施放時亮起的技能（例如：武公）。

#### 設置步驟
1. 將常駐技能圖示放置遊戲右下角的快捷鍵顯示介面上。
2. 點擊「Gen Img」按鈕。
3. 在路徑 `\DanceGPT\Image_customer\可用常駐技能` 中選擇常駐技能的圖片，並將其移動到 `\DanceGPT\Image_customer` 目錄下。
4. 重啟 DanceGPT，確認成功讀取對應數字的圖檔。
5. 為每個圖檔設定對應的快捷鍵。
6. 設定技能效果的持續時間。

<br>

### 🚨 測謊偵測
#### 使用說明
目前僅偵測 `菲歐娜測謊` 和 `滑鼠跟蹤測謊`。

#### 常見問題
- **為何沒有偵測到測謊？**  
  兩個檢查點間隔應 < 10 秒，否則可能錯過檢查。

- **為何頻繁遇到測謊？**  
  必須使用 Arduino Leonardo，並增加腳本隨機度與複雜度。
  非危險時段（02:00 ~ 08:00）可以持續跳舞超過 12 小時。

- **凌晨較易測謊？**  
  凌晨 02:00–08:00 系統例行性測謊，應避免跳舞及調整腳本。

- **平常正常，今天卻收到測謊？**  
  解完測謊後重啟電腦，若無改善請隔日再試。

<br>

### 🔴 紅點偵測
當紅點數量 ≥ N 且超過 T 秒，會自動回村並停止腳本。

<br>

### 🗺️ 換圖結束
當偵測到地圖變更時，腳本將停止運行。
 
<br>

### ⏳ 執行時間
可設定腳本自動停止的時間，例如設定 `10 hr 10 min`，腳本會在上午 10:10 停止。

<br>

### 📝 腳本設定
#### 常見問題
- **為何會跳出「請將小地圖打開」的視窗？**  
  請依照「前置設定」的說明進行調整。

- **解輪後角色位置亂掉？**  
  請將 `角色位置重置的操作` 寫在腳本中，例如撞牆或衝刺。

- **腳本發出 `嘟嘟嘟` 聲音並趴下？**  
  這是防止角色卡在繩子上的機制。

- **`Image_Capture` 資料夾中的圖片用途？**  
  這些圖片記錄了遊戲狀態，用於問題分析。
