## DanceGPT 使用指南

### 必要設定：

1. **確認遊戲解析度設置為 1366x768。**
2. **確保小地圖置於遊戲視窗左上角並顯示地圖名稱（可參考影片），並且BUFF狀態列（遊戲視窗最上方的那一排）不被遮擋。**
3. **將 Windows 縮放比例設為 100%。這可以在 Windows 的「系統 - 顯示器 - 比例」中設置。**
4. **關閉所有遊戲外置聊天視窗。**

### 功能：

- **執行時間**
  - **使用說明**：
    - 可以設定一個時間，當到達時，腳本將自動停止。
    - 例如：設定為 10 hr 10 min，則腳本會在上午 10:10 停止。
- **換圖結束**
- **紅點偵測**
  - **使用說明**：
    - 下拉選單用於選擇紅點數量 (N)。
    - 輸入欄位設定超時秒數 (T)。
    - 當紅點數量達到或超過 N，且持續時間超過 T 時（回村）結束腳本。
- **測謊偵測**
  - **Q&A**：
  1. **為什麼出現菲歐娜卻沒有報警？**
     - 兩個檢查點之間的時間間隔建議不超過 15 秒，否則可能會錯過檢查點的測謊檢查。
  2. **為什麼頻繁出現測謊？**
     - 請使用 Arduino Leonardo，並設定為在非危險時段內跳舞，可以持續跳滿 12 小時以上。虛擬 (virtual) 幸運通常能夠持續 1 小時，運氣不好則可能在 10 分鐘內遇到測謊。
     - 增加腳本的隨機度（每顆按鍵的時間最好都隨機）和複雜度（增加各 script 的 case 數）。
  3. **為什麼白天正常，晚上會遇到測謊？**
     - 建議在凌晨 02:00 到 06:00 之間避免跳舞，也不要在此時調整腳本，此時系統測謊的概率較高（包括菲歐娜、輸入驗證碼）。
  4. **為什麼平常都正常，今天卻一直收到測謊？**
     - 請在解完測謊後重啟電腦並重新登入。
- **解地圖輪**
  - **Q&A**：
  1. **為什麼解輪成功後卻顯示 FAIL 並停止？**
     - 確保每次解輪成功後，DanceGPT UI 中的解輪秒數會歸零。
     - 確保地圖輪 CD 圖示未被遮擋，否則會被判定為解輪失敗，秒數會繼續累積。
     - 使用常駐技能 Gen_Img 檢查視窗位置是否有偏移（參見「必要設定」第 3 點）。
  2. **為什麼解輪的成功率很低？**
     - 在格蘭蒂斯中，解輪成功率通常可達 95%。如果是在私服中，可能圖片已被調整過。
- **輔助技能**
  - **使用說明**：
    - 適用於 CD 時間固定的技能或消耗品（例如：經驗券）。
    - 第一個數字代表技能的冷卻時間，即每次施放之間需間隔的時間。
    - 第二個數字為技能的後搖時間，即技能施放完畢後需要等待多久才能行動。
    - (每個按鍵均可單獨設定，並且在運作時會同時生效)

- **常駐技能**
  - **使用說明**：
    - 適用於技能圖示會在可施放時亮起的技能（例如：CD 時間不固定或會跳無冷卻時間的技能）
  - **設置步驟**：
    1. 將常駐技能圖示放置於快捷鍵位置。
    2. 點擊「Gen Img」按鈕。（如果產生的圖示有偏移，請參考必要設定進行調整）
    3. 在路徑 `\DanceGPT\Image_customer\可用常駐技能` 中選擇常駐技能的圖片，並將其移動到 `\DanceGPT\Image_customer` 目錄下。
    4. 重啟 DanceGPT，確認是否成功讀取對應數字的圖檔。
    5. 為每個圖檔設定對應的快捷鍵。
    6. 設定技能效果的持續時間。

### 下載與安裝：

1. **為什麼開啟 DanceGPT 時會閃退？**
   - 請以系統管理員身份啟動程式。
   - 確保防毒軟體將 DanceGPT 設為信任程式。

### 腳本設定：

1. **為什麼會跳出「請將小地圖打開（需顯示地圖名稱，並放置於遊戲視窗的左上角）。目前僅支援遊戲解析度為 1366x768 下運行。」的視窗？**
   - 請確認「必要設定」中的第 1 至 4 點。
2. **解輪後角色位置亂掉了怎麼辦？**
   - 請在腳本中設置角色位置重置操作，例如撞牆或在平台上衝刺。
3. **為什麼腳本運行中會發出嘟嘟嘟的聲音？**
   - 這是為了防止角色卡在繩子上。如果角色位置長時間沒有變化，會觸發這個聲音。
4. **Image_Capture 資料夾中的圖片有什麼用途？**
   - 這些圖片記錄了遊戲運行的狀態，以便在出現問題時進行分析。

### Arduino：

1. **我可以使用 Arduino Uno 或 Arduino Mega 嗎？**
   - 不可以！必須使用 Arduino Leonardo，建議在蝦某、某皮等網站購買約 200 多元的 Arduino Leonardo。
2. **為什麼顯示 avrdude.exe 或 avrdude.conf 路徑錯誤？**
   - 確認路徑中沒有中文字符。
   - 使用 Arduino 預設的安裝路徑，理論上只需修改 UserName。
3. **為什麼重啟 Arduino Leonardo 後沒有看到 bootloader？**
   - 開啟系統音效，確認是否有插入或拔出的音效提示。
   - 在裝置管理員中，重啟後立即點擊「掃描硬體變更」。
4. **為什麼選擇 Arduino 時會跳回 Virtual？**
   - 可能是因為在程序運行到一半才切換，導致 USB 連線不穩。
   - 啟動時就進行切換，Arduino 可能沒有成功燒錄或 COM port 選擇錯誤。
5. **換角色或電腦時，Arduino 需要重新燒錄嗎？**
   - 不需要。
