## 📘 使用指南

### 🧰 安裝
1. **第一次使用**：請先執行 `setup` 完成安裝。  
   - 可參考 [YouTube 教學影片](https://www.youtube.com/watch?v=k2DSwJA6xig) 進行下載與安裝。

<br>

### 💻 開啟程式
1. **防毒與廣告攔截軟體設定**  
   - 避免 Windows 自動刪除（Windows 11）：至「病毒與威脅防護設定」->「新增排除項目」-> 將 DanceGPT 根資料夾新增為排除範圍。   
   - 將 **DanceGPT** 設為防毒軟體信任程式。  
   - 將任何廣告攔截軟體設為允許 **DanceGPT** 運行。

2. **正確開啟程式**  
   - 請雙擊原下載的解壓縮包中的 `DanceGPT.bat` 以開啟程式。
   - **勿使用 `setup` 安裝路徑中的檔案** 開啟程式。  
   - 不再需要系統管理員權限開啟，且 **勿直接點擊 `TySyLdAsTsOpMaAcReSx.exe`**。

<br>

### ⬆️ 更新程式
1. **更新方式**
   - 下載最新版本後，**解壓縮後直接使用新版資料夾**中的 `DanceGPT.bat` 開啟程式。**勿使用舊資料夾或覆蓋舊檔**。
   - 若需保留舊設定，可將以下檔案從舊版本資料夾移至新版本資料夾：
     - `UserData` 資料夾  
     - `Script_main.m`  
     - `Script_body.m`

### 🖥️ 畫面設置
1. 🟥**Windows 縮放比例**：設為 `100%`  
   - 路徑：`設定 > 系統 > 顯示器 > 比例`，設置後請重啟電腦

2. 🟧**遊戲解析度**：設為 `1366x768`

3. **小地圖與 BUFF 顯示**：  
   - 🟨小地圖置於畫面 **第二象限 (左上區域)**，並顯示地圖名稱  
   - 🟩關閉 **`BUFF 自動排列`**，確保 BUFF 狀態列可見
   - 🟩關閉 **`戰鬥時套用UI透明度`**，確保腳本運行

4. 🟦 **`Lv.` / `角色ID` / `選單`**：畫面清楚可見，不被其他視窗遮擋

5. 🟪**聊天視窗**：  
   - 關閉外置聊天視窗
   - 選擇 `全體` 頻道，開啟聊天室窗 `呈現多行` 訊息（**偵測 GM 必要**）

<br>

### 🔌 硬體要求 — Arduino Leonardo
- 僅支援 **`Leonardo`**（Uno / Mega 不支援）  
- 可於蝦皮、某皮等網站購買，價格約 **200~300 元**

<br>

### 💬 常見問題
- **換角色 / 腳本 / 電腦時，Arduino 需重新燒錄嗎？**  
  - 不需要!

- **為什麼按下 Dance 時 UI 跳回 Virtual？**  
  - Arduino 未成功燒錄或 COM port 選錯  
  - 跳舞過程中途跳回 Virtual，可能是 USB 連接不穩或接觸不良

- **為什麼顯示 avrdude.exe 或 avrdude.conf 路徑錯誤？**  
  - 路徑不能含中文字符。建議使用 Arduino 預設安裝路徑，只需修改 ..\UserName\..

- **為什麼重啟 Arduino Leonardo 後看不到 bootloader？**  
  - 開啟系統音效確認插拔提示
  - 開啟裝置管理員，重啟後立即點擊「掃描硬體變更」

### 🗡️ 腳本直用
#### 使用說明
職業與等級符合即可無腦使用，經長時間測試能穩跳 6 小時以上。

#### 設置步驟
1. 從 Dance UI 的公告區查看下載連結。

<p align="left">
<img src="https://github.com/user-attachments/assets/f19f75ea-611f-49fe-9182-d2073b8a4d4b" alt="Direct_Script_DL_link" width="302" height="102">
</p>

2. 挑選對應職業與等級區間的地圖，下載對應腳本。
3. 放入挑選的職業等級腳本與設定檔：
   - `Direct_Script_main.m` 和 `Direct_Script_body.m` 放入 Dance 資料夾。
   - `Direct_setting.txt` 放入 `UserData` 資料夾。
4. 設置遊戲快捷鍵：根據 `KeyboardShortcuts.png` 配置設置。
6. 設定常駐技能（若有）：
   - 根據 `Direct_setting.txt`，確保常駐技能的快捷鍵出現在遊戲右下角的快捷鍵顯示介面上。 
   - 使用 `Gen_Img` 生成技能圖檔並放入 `Image_customer` 資料夾。(可參考常駐技能-設置步驟)
7. 啟動腳本，開始快樂跳舞！

<br>

### 🧙‍♂️ 解地圖輪
#### 常見問題
- **解輪後顯示 FAIL 並停止**：檢查地圖輪秒數是否歸零，BUFF列是否遮擋。請依照「前置設定」的說明進行調整。
- **解輪成功率低**：格蘭蒂斯成功率可達 95%，私服可能有圖像調整，導致無法解輪。

<br>

### 🪄 輔助技能
#### 使用說明
適用於冷卻時間固定的技能或消耗品（例如：經驗券）。  
第一個數字代表技能的冷卻時間，即每次施放之間需間隔的時間。  
第二個數字為技能的後搖時間，即技能施放完畢後需要等待多久才能行動。  
每個按鍵均可單獨設定，並且在運作時會同時生效。

<br>

### 🛡️ 常駐技能
#### 使用說明
適用於技能圖示會在可施放時亮起的技能（例如：武公）。

#### 設置步驟
1. 將常駐技能圖示放置遊戲右下角的快捷鍵顯示介面上。
2. 點擊「Gen Img」按鈕。
3. 在路徑 `\DanceGPT\Image_customer\可用常駐技能` 中選擇常駐技能的圖片，並將其移動到 `\DanceGPT\Image_customer` 目錄下。
4. 重啟 DanceGPT，確認成功讀取對應數字的圖檔。
5. 為每個圖檔設定對應的快捷鍵。
6. 設定技能效果的持續時間。

<br>

### 🚨 測謊偵測
#### 使用說明
目前僅偵測 `菲歐娜測謊` 和 `滑鼠跟蹤測謊`。

#### 常見問題
- **為何沒有偵測到測謊？**  
  兩個檢查點間隔應 < 10 秒，否則可能錯過檢查。

- **為何頻繁遇到測謊？**  
  必須使用 Arduino Leonardo，並增加腳本隨機度與複雜度。
  非危險時段（02:00 ~ 08:00）可以持續跳舞超過 12 小時。

- **凌晨較易測謊？**  
  凌晨 02:00–08:00 系統例行性測謊，應避免跳舞及調整腳本。

- **平常正常，今天卻收到測謊？**  
  解完測謊後重啟電腦，若無改善請隔日再試。

<br>

### 🔴 紅點偵測
當紅點數量 ≥ N 且超過 T 秒，會自動回村並停止腳本。

<br>

### 🗺️ 換圖結束
當偵測到地圖變更時，腳本將停止運行。
 
<br>

### ⏳ 執行時間
可設定腳本自動停止的時間，例如設定 `10 hr 10 min`，腳本會在上午 10:10 停止。

<br>

### 📝 腳本設定
#### 常見問題
- **為何會跳出「請將小地圖打開」的視窗？**  
  請依照「前置設定」的說明進行調整。

- **解輪後角色位置亂掉？**  
  請將 `角色位置重置的操作` 寫在腳本中，例如撞牆或衝刺。

- **腳本發出 `嘟嘟嘟` 聲音並趴下？**  
  這是防止角色卡在繩子上的機制。

- **`Image_Capture` 資料夾中的圖片用途？**  
  這些圖片記錄了遊戲狀態，用於問題分析。
