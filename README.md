## 📘 使用指南

### 🧰 安裝
1. **第一次使用**：請先執行 `setup` 完成安裝。  
   - 可參考 [YouTube 教學影片](https://www.youtube.com/watch?v=k2DSwJA6xig) 進行下載與安裝。

<br>

### 💻 開啟程式
1. **防毒與廣告攔截軟體設定**  
   - 避免 Windows 自動刪除（Windows 11）：至「病毒與威脅防護設定」->「新增排除項目」-> 將 DanceGPT 根資料夾新增為排除範圍。   
   - 將 **DanceGPT** 設為防毒軟體信任程式。  
   - 將任何廣告攔截軟體設為允許 **DanceGPT** 運行。

2. **正確開啟程式**  
   - 請雙擊原下載的解壓縮包中的 `DanceGPT.bat` 以開啟程式。
   - **勿使用 `setup` 安裝路徑中的檔案** 開啟程式。  
   - 不再需要系統管理員權限開啟，且 **勿直接點擊 `TySyLdAsTsOpMaAcReSx.exe`**。

<br>

### ⬆️ 更新程式
1. **更新方式**
   - 下載最新版本後，**解壓縮後直接使用新版資料夾**中的 `DanceGPT.bat` 開啟程式。**勿使用舊資料夾或覆蓋舊檔**。
   - 若需保留舊設定，可將以下檔案從舊版本資料夾移至新版本資料夾：
     - `UserData` 資料夾  
     - `Script_main.m`  
     - `Script_body.m`

### 🖥️ 畫面設置
1. 🟥**Windows 縮放比例**：設為 `100%`  
   - 路徑：`設定 > 系統 > 顯示器 > 比例`，設置後請重啟電腦

2. 🟧**遊戲解析度**：設為 `1366x768`

3. **小地圖與 BUFF 顯示**：  
   - 🟨小地圖置於畫面 **第二象限 (左上區域)**，並顯示地圖名稱  
   - 🟩關閉 **`BUFF 自動排列`**，確保 BUFF 狀態列可見
   - 🟩關閉 **`戰鬥時套用UI透明度`**，確保腳本運行

4. 🟦 **`Lv.` / `角色ID` / `選單`**：清楚可見，不被其他視窗遮擋

5. 🟪**聊天視窗（偵測 GM 必要）**：  
   - 關閉外置聊天視窗
   - 將對話框字型大小設為 `基本`，並 `隱藏聊天時間` 
   - 選擇 `全體` 頻道，開啟聊天室窗 `呈現多行` 訊息

<br>

- 僅支援 **`Leonardo`**（Uno / Mega 不支援）  
- 可於蝦皮、某皮等網站購買，價格約 **200~300 元**

### 💬 常見問題
- **換角色 / 腳本 / 電腦時，Arduino 需重新燒錄嗎？**  
  - 不需要!

- **為什麼按下 Dance 時 UI 跳回 Virtual？**  
  - Arduino 未成功燒錄或 COM port 選錯  
  - 跳舞過程中途跳回 Virtual，可能是 USB 連接不穩或接觸不良

- **為什麼顯示 avrdude.exe 或 avrdude.conf 路徑錯誤？**  
  - 路徑不能含中文字符。建議使用 Arduino 預設安裝路徑，只需修改 ..\UserName\..

- **為什麼重啟 Arduino Leonardo 後看不到 bootloader？**  
  - 開啟系統音效確認插拔提示
  - 開啟裝置管理員，重啟後立即點擊「掃描硬體變更」

4. 重啟 DanceGPT，確認已讀取對應數字的圖檔。
5. 為每個圖檔設定快捷鍵。
7. 設定技能效果的持續時間

#### 常見問題

|------|------------|

### 📬🐕‍🦺 Discord 看門狗 - 例行性/緊急通報
- 出門在外也能查看角色狀態  
- 緊急事件 → 高亮通知  
⚠️ iOS 可能無法即時通知，需將 **電腦版 Discord 關閉** 才能正常推播

#### 設定方法
<img width="449" height="746" alt="image" src="https://github.com/user-attachments/assets/e4f80e2a-1374-45f2-934e-4db9b07f7e1f" />

1. **步驟 0**：建立私人 Discord 伺服器（專門接收通知）
2. **步驟 1–3**：開啟 **開發者模式**  
   - 路徑：`⚙️設定 > 進階 > 開發者模式`
3. **步驟 4–10**：建立 Webhook  
   - 選擇頻道 → `編輯頻道 > 整合 > Webhook > 新增`  
   - 輸入名稱（例：Captain Hook）  
   - 選擇要綁定的頻道（例：`#文字頻道-公開`）  
   - 複製 **Webhook URL**
4. **步驟 12–13**：取得使用者 ID  
   - 在 Discord 中右鍵點擊頭像 → **複製使用者 ID**
5. **步驟 11 & 14**：建立 `Discord_WatchDog.txt`  
   - 路徑：`UserData` 資料夾  
   - 內容範例：
     ```txt
     Webhook_URL=你的Webhook網址
     USER_ID=你的使用者ID
     ```
   ⚠️ 檔名必須正確為 `Discord_WatchDog.txt`，否則無法讀取  

✅ 完成後，DanceGPT 即可透過 Discord Webhook 推送角色狀態與通知
